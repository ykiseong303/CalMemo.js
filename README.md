# CalMemo_react.js

## 프로젝트 명 : 칼메모
+ 칼메모는 Calendar + Memo의 합성어로,  
메모한 내용을 캘린더에 다시 정리해야하는 번거로움을 해결하기 위해  
메모와 동시에 캘린더에 자동으로 입력되는 웹/앱입니다.
